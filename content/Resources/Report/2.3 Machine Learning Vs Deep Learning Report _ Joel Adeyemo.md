

## Traditional Machine Learning
 Example: Email Spam Detection 
 - Algorithm : Logistic Regression or Decision Tree

### Real World Application:
  Email providers like Gmail, Proton mail or Outlook use the spam filter to remove unwanted emails and keep user email from cluster.
  
### Why  Machine Learning is Suitable:
- Logic regression works well because spam detection uses structured features such as frequency of words, presence of links or sender's address
- Machine Learning allows for classification of dataset outcome ie (spam or not spam\)
- Machine Learning is fast and requires less computer power and performs well with moderate dataset

### Why Deep Learning would not Work effectively:
   - Deep Learning requires large amount of data and computing resource 
   - Spam detection often depends on a limited number of well-defined features
   - Since deep learning uses neural networks with complex  hidden layers "deep" then the complexity would increase without providing improvement compares to ML.


## Deep Learning Example
 Example: Autonomous Driving 
 - Algorithm : Deep Reinforcement Learning or CNNs

### Real World Application:
  Driving Companies like Tesla and Waymo, have self driving cars that use deep learning to detect pedestrian, traffic signs and other vehicles and making real time driving decision.Also companies like comma ai, use deep neural networks(Tinygrad) to enable semi-self driving on normal cars by using cameras to do the same thing.
  
### Why  Deep Learning is Suitable:
- Autonomous driving requires processing of large amount of complex data from camera, radar and sensors
- The large amount of data enables CNN(Convolutional neural networks) and deep learning models to recognize object and make decision in real time


#### Why Machine Learning would not Work effectively:
   - Machine learning requires manually selecting feature ie data characteristic which is not may not work for really broad driving conditions and situation. In driving there are so many scenarios that are also dependent on other users action which Machine learning cannot account for such as road, weather, lighting m traffic, unexpected object etc.
