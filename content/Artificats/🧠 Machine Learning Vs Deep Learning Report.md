## Introduction

This project is about understanding the relationship between Machine Learning and Deep Learning within the field of Artificial intelligence. It allows for the understanding of the different techniques and architectures with Machine Learning encompassing a broader range of algorithms and Deep learning focusing specifically on artificial neural networks.

## Objective
 The primary objective of this report is to provide a comprehensive comparative analysis of Machine learning  and Deep Learning specifically:
 - Exploring the application where each methodology is effective
 - Examine technical foundations and requirement of each approach
